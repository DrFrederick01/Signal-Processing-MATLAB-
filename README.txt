This folder contains all the code for data processing, with
"given_swallow_analysis.m" analysing the provided dataset, and
"recorded_swallow_analysis.m" analysing our own recordings.

Output graphs can be found in "output/", and are sorted by the given data
and our recorded data. Audio files are located in "audio/".

These files were created and verified to be working in MATLAB R2024b.

Dependencies:
- Signal Processing Toolbox
